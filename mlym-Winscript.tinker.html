User-Configurable Options:

The following options are currently configured as shown:
  • Allow chandrakkala ( ് ) to be typed even after vowels: 〔ViramaAnywhere〕
  • Use Malayalam digits rather than Western digits by default: 〔ScriptDigit〕
  • Enable use of left and right quote marks: 〔SmartQuotes〕

Conjunct characters:

•    A simple conjunct  is typed by pressing  〔JOIN〕  between the consonants.          e.g.  〔NA〕〔JOIN〕〔NA〕  ➔  ന്ന
•    A joined conjunct is typed by pressing  〔JOIN〕〔JOIN〕  between the consonants.        e.g.  〔NA〕〔JOIN〕〔JOIN〕〔NA〕  ➔  ന്‍ന
•    A non-joined conjunct is typed by pressing  〔NONJOIN〕   between the consonants.     e.g.  〔NA〕〔NONJOIN〕〔NA〕  ➔  ന്‌ന

Alternative: To change one type of conjunct to another, press the 〔TOGGLE〕  key after it.
        e.g.  ന്ന   〔TOGGLE〕  ➔ ന്‍ന   〔TOGGLE〕  ➔ ന്‌ന   〔TOGGLE〕  ➔ ന്ന

Dependent/Independent Vowels:

The keyboard will normally produce independent vowels, except after consonants, when it will produce independent vowels. 
    e.g.        〔I〕  ➔ ഇ              〔GA〕〔I〕  ➔ ഗി              
To force an independent vowel after a consonant, press  〔A〕 after the consonant.      e.g.      〔GA〕〔A〕〔I〕  ➔ ഗഇ              
Alternative: To change one type of vowel to the other, press the 〔TOGGLE〕  key after it.    e.g. ഗഇ   〔TOGGLE〕  ➔ ഗി   〔TOGGLE〕  ➔ ഗഇ

Alternate forms:

In addition to altering conjuncts and vowels, the 〔TOGGLE〕  key can produce changes like these:
    e.g.     ൪   〔TOGGLE〕  ➔ 4      toggle digits: script ➔ roman
        "   〔TOGGLE〕  ➔ “          toggle smart quotes on/off

Alternate characters:

Various related characters can be obtained by pressing the same key again without pausing.
    e.g.      〔NA〕  ➔ ന   〔NA〕  ➔ ണ   〔NA〕  ➔ ഞ   〔NA〕  ➔ ങ
If you actually want a sequence of the character, either pause or press  〔A〕  in between.
    e.g.     〔NA〕〔NA〕  ➔ ണ           〔NA〕〔A〕〔NA〕  ➔ നന  

Smart Quotes:

Press the single quote   '  or double quote  "  key again to switch between left and right smart quotes.  e.g.    “   "  ➔ ”

Brackets:

Press the open or close parenthesis key up to 3 more times to obtain alternative forms of bracketing.
    e.g.    (   (  ➔ [   (  ➔ {   (  ➔ ⌊ 

Note to Advanced Users:

If you need an independent ZWJ or ZWNJ character (very rare), type  space  followed by  〔JOIN〕  or  〔NONJOIN〕  respectively.
